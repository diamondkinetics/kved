19/* kved structure
20
21<- 4 bytes -><- 4 bytes -> <= 8 bytes when using flash with word of 64 bits
22+------------+------------+
23| SIGNATURE | COUNTER | <= HEADER ID AND NEWER COPY IDENTIFICATION
24+---------+--+------------+
25|KEY ENTRY|TS| KEY VALUE | <= VALID KEY (KEY ENTRY, TYPE, SIZE AND VALUE)
26+---------+--+------------+
27|KEY ENTRY|TS| KEY VALUE |
28+---------+--+------------+
29|KEY ENTRY|TS| KEY VALUE |
30+---------+--+------------+
31|000 ... 0000| KEY VALUE | <= ERASED KEY
32+------------+------------+
33|FFF ... FFFF|FFF ... FFFF| <= EMPTY ENTRY
34+------------+------------+
35|FFF ... FFFF|FFF ... FFFF|
36+------------+------------+
37|FFF ... FFFF|FFF ... FFFF|
38+------------+------------+
39
40*/
